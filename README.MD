### Cara Penggunaan Template

Silahkan cek file satu persatu. Terdapat komentar di setiap file, untuk mempermudah kalian dalam ngoding. \
Untuk Kodingan bagian tampilan kalian bisa nyari (ctrl + f) cari `{kode kamu}` \
Silahkan ubah aja bagian itu sesuai apa yang kalian pelajar.

Enjoy codingnya ^-^
